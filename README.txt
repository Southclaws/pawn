Instructions:

1) download http://www.compuphase.com/pawn/pawn-3.2.3664.zip
2) unzip pawn-3.2.3664.zip somewhere, e.g. C:\pawn-3.2.3664
3) clone this repo: git clone git://github.com/Zeex/pawn-3.2.3664_patches.git
4) cd to pawn-3.2.3664_patches
5) in your linux/cygwin/msys/whatever shell, run ./apply_all.sh C:\pawn-3.2.3664 (this step assumes you have the "patch" utility installed)
6) finally, build the compiler
