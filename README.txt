Instructions:

1) unzip pawn-3.2.3664.zip somewhere, e.g. C:\pawn-3.2.3664
2) clone this repo: git clone git://github.com/Zeex/pawn-3.2.3664_patches.git
3) cd to pawn-3.2.3664_patches
4) in your linux/cygwin/msys/whatever shell, run ./apply_all.sh C:\pawn-3.2.3664 (this step assumes you have the "patch" utility installed)
5) finally, build the compiler
